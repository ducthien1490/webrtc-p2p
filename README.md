WebRTC-P2P
===
This is a sample applicaiton of Web RealTime Communication running in Peer-to-peer method (P2P)

Install
---
- Requirements: Nodejs
- Install dependencies:
  - ```cd app/rtc```
  - ```npm install```

Usage
---
- Start Rails server: ```rails s```
- Start node server: ```cd app/rtc && node server.js```
- Enjoy!
